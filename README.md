1. Project Overview

This repository contains the dataset used in the thesis “Evaluation of Usability and User Behavior in Public and Private Sector Digital Services in Greece Using Multimodal Biometric Measurements”, evaluating usability 
and user behavior in public and private digital services in Greece using a multimodal measurement approach.

2. Dataset

File: master_dataset.csv

The dataset includes all participant-level observations across tasks (A1–T3) and platforms.

3. Data Content (All Indicators)

ParticipantID
Task
Platform
CompletionTime
SuccessRate / Status
Cognitive Indicators
CognitiveLoad (Mean)
CognitiveLoad (Peak)
EDA (Electrodermal Activity)
HR (Heart Rate)
HR Delta (change from baseline)
Confusion (Mean)
FixationCount
FixationDuration
GazeValidRatio

4. Data Structure
   
Each row represents a participant–task observation
Tasks are coded as A1–T3:
Letter = platform
Number = specific task

5. Missing Values

Missing values indicate that data were not available or not recorded for specific observations.

6. Usage

The dataset supports reproduction of all analyses presented in the thesis, including task-level, platform-level, and sector-level evaluation.
